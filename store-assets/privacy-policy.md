# Privacy Policy

Effective date: June 6, 2026

ChatGPT Tool Auto Approver is a local browser extension for automatically approving trusted ChatGPT tool confirmation prompts.

## Data Collection

This extension does not collect, transmit, sell, share, or store user data outside the user's browser.

## Local Page Access

The extension runs only on `chatgpt.com` and `chat.openai.com`. It reads visible page text and button labels locally to detect ChatGPT tool confirmation cards. This local inspection is used only to identify an Allow/Approve button inside a guarded confirmation prompt.

## Settings

The extension stores user preferences in Chrome extension storage, such as whether auto-approval and stall refresh are enabled. These settings remain local to the browser profile unless Chrome sync is enabled by the user.

## Third Parties

The extension does not send any data to the developer, analytics services, advertising networks, or other third parties.

## Contact

For questions about this extension, contact the developer account used for the Chrome Web Store listing.
